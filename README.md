# PBI Product Analysis

Repo progetto Menarini dedicata ai consumer `PBIP`, report e semantic model.

## Contenuto

- `powerbi-projects`
  Progetti Power BI versionati del perimetro Menarini.

## Esclusioni intenzionali

Questa repo non contiene:

- framework `modularity`
- framework `repository-health`
- sorgenti grezze e assessment di migrazione

Gli asset locali transitori come `module-config`, log, snapshot ed evidence di test non sono parte del source repository.
